# 사진 업로드 / GPS / 제출 Preview 스모크 테스트

## 테스트 원칙

- Production 환경에서는 사진 업로드, GPS 취득, 최종 제출 자동 테스트를 실행하지 않는다.
- Preview URL에서만 실행한다.
- `TEST03` 전용 조사원 계정과 `TEST-*` 표본만 사용한다.
- 실제 농가 개인정보, 실제 사진, 실제 제출 데이터는 사용하지 않는다.
- 운영 S01~S08 조사원 및 실제 표본에는 영향을 주지 않는다.
- 운영 데이터가 생성될 수 있는 테스트는 실행 직전에 사용자 확인을 받는다.

## 확인 범위

1. `TEST03` 로그인
2. `TEST-*` 표본만 조회되는지 확인
3. TEST 표본에서 사진 1장 업로드
4. GPS 취득 및 accuracy 저장 확인
5. 필수 입력값 작성 후 제출
6. 제출 후 상태값 변경 확인
7. `photo_log`, `gps_log`, `submissions` 또는 Google Sheets 저장 위치가 TEST 표본 기준인지 확인
8. Google Drive 저장 파일명이 TEST prefix이거나 TEST 폴더에 저장되는지 확인
9. 운영 S01~S08 조사원 및 실제 표본에는 영향이 없는지 확인

## 준비 항목

- Preview 환경변수는 Production과 동일한 구조로 설정하되, 가능하면 테스트 Sheet/Drive를 사용한다.
- 테스트 표본은 `sample_master`에 `TEST-*` prefix로 준비한다.
- 테스트 계정은 `TEST03`만 사용한다.
- TEST 표본의 `surveyor_id`는 `TEST03`으로 지정한다.
- Google Drive는 TEST 전용 폴더 또는 TEST prefix 파일명 정책을 사용한다.

## 테스트 시나리오

### 1. 로그인 및 표본 목록

- Preview URL 접속.
- `TEST03`으로 로그인.
- 표본 목록에 `TEST-*` 표본만 표시되는지 확인.
- S01~S08 운영 조사원 표본이 섞여 나오면 테스트를 중단한다.

### 2. TEST 표본 상세 화면

- `TEST-*` 표본 상세 화면에 진입한다.
- 조사 유형 선택, 표본 정보, 사진 업로드, GPS 증빙, 제출 버튼이 표시되는지 확인한다.
- 실제 농가명, 실제 연락처, 실제 주소가 테스트 화면에 표시되면 테스트를 중단한다.

### 3. 사진 업로드

- 테스트용 이미지 1장을 업로드한다.
- Drive 원본 저장이 완료되는지 확인한다.
- 업로드 응답 또는 기록에 Drive file id가 생성되는지 확인한다.
- 저장 위치가 TEST 폴더이거나 파일명이 TEST prefix인지 확인한다.
- 실제 현장 사진은 사용하지 않는다.

### 4. GPS 취득

- 위치 권한을 허용한다.
- 앱 GPS 위도/경도와 accuracy 값이 표시되는지 확인한다.
- `gps_log`에 TEST 표본 기준으로 기록되는지 확인한다.
- Production에서는 GPS 권한 테스트를 하지 않는다.

### 5. 제출

- 필수 입력값을 TEST 데이터로 작성한다.
- 사진 업로드와 GPS 취득이 완료된 상태에서만 제출한다.
- 제출 성공 후 `survey_submissions`, `survey_answers`, `gps_log`, `media_files` 또는 운영 중인 Google Sheets 저장 위치에 TEST prefix 기준 행이 생성되는지 확인한다.
- 제출 후 상태값이 TEST 표본 기준으로만 변경되는지 확인한다.

### 6. 격리 및 정리

- 테스트 완료 후 생성된 TEST 사진, GPS 로그, 제출 데이터는 삭제한다.
- 삭제하지 않는 경우 `isTest=TRUE`, `TEST_*`, `test_closed` 등으로 명확히 격리한다.
- TEST 데이터 정리 결과를 QA 기록에 남긴다.

## 중단 조건

- Production URL에서 테스트 중임을 발견한 경우.
- TEST03이 아닌 S01~S08 계정으로 로그인한 경우.
- `TEST-*`가 아닌 실제 표본이 목록에 표시된 경우.
- 실제 농가 개인정보 또는 실제 현장 사진을 사용해야 하는 상황이 된 경우.
- 저장 위치가 TEST 폴더 또는 TEST prefix 정책을 만족하지 않는 경우.
