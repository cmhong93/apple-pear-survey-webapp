# 조사원 교육용 화면 점검표

## 로그인

- 관리자 계정은 `/admin` 접근 가능
- 조사원 계정은 본인 표본 화면 접근 가능
- 비로그인 상태에서 관리자 화면은 조사 화면으로 이동
- 비밀값, 토큰, 개인정보가 오류 메시지에 표시되지 않음

## 표본 리스트

- 표본 ID, 품목, 품종, 행정구역, 조사상태 표시
- 생육 해당 여부 컬럼 표시
- 표본 선택 시 상단 공통 정보 카드 갱신
- 표본 변경 시 기존 입력값 혼합 방지

## 농가기본정보

- 조사일이 첫 항목으로 표시
- 표본 ID와 품종은 표본리스트 자동값으로 읽기전용 표시
- 과수 세부 품종은 자유입력이며 품종별 예시 안내 표시
- 재식거리 입력 후 자동 재식주수 안내 표시
- 개화시작일/만개기 평년은 `MM-DD` 형식 입력
- 저온피해 비율은 0~100 범위로 검증

## 사진/GPS

- 농가기본 사진 4종 표시
- 전경 사진 1, 전경 사진 2, MYGPS-660 사진, 계좌입금의뢰서 사인본 필수
- 계좌입금의뢰서 사인본은 AI 판독 제외 및 관리자 확인 대상
- GPS 수집 상태와 좌표 정합성 안내 표시

## 제출/PDF

- 필수 사진 미업로드 시 제출 차단
- 제출 성공 시 submission_id 표시
- 공식 PDF 템플릿 기반 PDF 생성 버튼 동작
- PDF 파일명에 농가명, 전화번호, 주소, 계좌정보 미포함
- export_manifest 기록 확인

## 관리자 화면

- 대시보드 카드 표시
- 사진/GPS 메뉴 표시
- PDF 산출물 메뉴 표시
- 관리자 검토 메뉴 표시
- 개인정보 원문은 권한 있는 화면에서만 표시
